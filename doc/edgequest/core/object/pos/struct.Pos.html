pub struct Pos {
    pub x: isize,
    pub y: isize,
}

Hold an x, y cartesian coordinate. Should be used when explicitly needing to manipulate points, not just store them. Length and width should be stored as tuples, however a monster should be a pos to calculate distance between it and the player.

x - x axis location y - y axis location
